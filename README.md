# asdf-soracom

[soracom-cli](https://github.com/soracom/soracom-cli) plugin for [asdf](https://github.com/asdf-vm/asdf) version manager

## Installation

Install the plugin:

```
asdf plugin add soracom https://github.com/grimoh/asdf-soracom.git
```
